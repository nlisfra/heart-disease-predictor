# 🫀 Heart Disease Predictor

Aplikasi prediksi risiko penyakit jantung menggunakan Machine Learning.

## Demo
[Link Streamlit App](https://heart-disease-predictor-h3nohku9tawcucb8gzhgrw.streamlit.app)

## Tech Stack
- Python
- Scikit-learn (Random Forest)
- Streamlit

## Cara Menjalankan
pip install -r requirements.txt
streamlit run app.py
